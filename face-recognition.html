Face Recognition & Surveillance Cam 🔒

A facial recognition-based surveillance system for smart security applications — enhancing safety and monitoring efficiency.

Project Overview

The Face Recognition & Surveillance Cam is designed to provide smart security monitoring. Using AI-based facial recognition, real-time alerts, and automated logging, it ensures efficient surveillance for homes, offices, and sensitive areas.

Highlights 🌟

  • 🤖 AI Models: OpenCV & FaceNet for accurate facial detection and recognition.
  • 📹 Real-Time Monitoring: Live camera feed with event-triggered recording.
  • 🔔 Alerts: Instant notifications on recognized/unrecognized faces via email or mobile app.
  • 💾 Logging: Maintains a database of detected faces with timestamps for audit purposes.
  • 🌐 Connectivity: Web and mobile dashboard for remote access and monitoring.
  • 🔧 Prototype Level: Functional prototype ready for field testing.
